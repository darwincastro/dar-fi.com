## Cisco Platform

IOS XE

## CLI

```
show interface
```

## WLC's version used

17.6.2

## Yang Module

Cisco-IOS-XE-interfaces-oper

## Prefix

interfaces-ios-xe-oper


## XPath


/interfaces-ios-xe-oper:interfaces/interface


## Example:

### GET

```json
curl -X 'GET' \
  'http://10.93.178.145:8480/restconf/proxy/https://10.93.178.68:443/restconf/data/Cisco-IOS-XE-interfaces-oper:interfaces/interface' \
  -H 'accept: application/yang-data+json'
```

### Sending

```json
http://10.93.178.145:8480/restconf/proxy/https://10.93.178.68:443/restconf/data/Cisco-IOS-XE-interfaces-oper:interfaces/interface
```
