## Cisco Platform

IOS XE

## CLI

```CLI
copy bootflash:test_copy_file.txt harddisk:copied.txt
```

## WLC's version used

17.9.1ES13

## Yang Module

Cisco-IOS-XE-rpc

## Prefix

ios-xe-rpc

## XPath

ios-xe-rpc:copy

## Example:
### RPC

```xml
<rpc xmlns="urn:ietf:params:xml:ns:netconf:base:1.0" message-id="101">
  <copy xmlns="http://cisco.com/ns/yang/Cisco-IOS-XE-rpc">
    <source-drop-node-name>bootflash:test_copy_file.txt</source-drop-node-name>
    <destination-drop-node-name>harddisk:copied.txt</destination-drop-node-name>
  </copy>
</rpc>
```

### Received message from host

```xml
<rpc-reply xmlns="urn:ietf:params:xml:ns:netconf:base:1.0" xmlns:nc="urn:ietf:params:xml:ns:netconf:base:1.0" message-id="urn:uuid:809fe9f4-a955-45a9-a94c-32d0a509ecd7"><result xmlns="http://cisco.com/ns/yang/Cisco-IOS-XE-rpc">RPC request successful</result>
</rpc-reply>
```
