## Cisco Platform

C9800-CL

## WLC's version used

17.6.2

## Yang Module

Cisco-IOS-XE-wireless-access-point-oper

## Prefix


/wireless-access-point-oper


## XPath


/wireless-access-point-oper:access-point-oper-data/capwap-data


## Example:
### Sending

```json
http://<IP:PORT>/restconf/proxy/https://<IP:PORT>/restconf/data/Cisco-IOS-XE-wireless-access-point-oper:access-point-oper-data/capwap-data=<AP MAC>/country-code
```

### Received message from host

```json
{
  "Cisco-IOS-XE-wireless-access-point-oper:country-code": "US "
}
```
