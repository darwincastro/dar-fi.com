import Note from '../../components/common/Note.astro';

This will apply **Only** for physical 9800-40, 9800-80 and 9800-L "on the Wireless side" it should work with other Cisco Catalyst devices like CAT9300-9500, etc.

<Note>
Please refer to the following <a href="https://dar-fi.com/blog/ios-xe_tracking_xpaths" target="_blank">post</a> to learn how to track XPaths.
</Note>


## Cisco Platform

IOS XE

## CLI

```CLI
show environment summary
```

## WLC's version used

17.9.1ES13

## Yang Module

Cisco-IOS-XE-environment-oper

## Prefix

environment-ios-xe-oper

## XPath

/environment-ios-xe-oper:environment-sensors/environment-sensor

## Example:
### Sending REST

```json
http://10.93.178.145:8480/restconf/proxy/https://10.93.178.68:443/restconf/data/Cisco-IOS-XE-environment-oper:environment-sensors/environment-sensor={environment-sensor-name},location={environment-sensor-location}
```
