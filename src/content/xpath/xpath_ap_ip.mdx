## Cisco Platform

C9800-CL

## WLC's version used

17.6.2

## Yang Module

Cisco-IOS-XE-wireless-access-point-oper

## Prefix


/wireless-access-point-oper


## Xpath


/wireless-access-point-oper:access-point-oper-data/capwap-data


## Example:

### Sending

```json
http://<IP:PORT>/restconf/proxy/https://<IP:PORT>/restconf/data/Cisco-IOS-XE-wireless-access-point-oper:access-point-oper-data/capwap-data=00%3A04%3A05%3A00%3A02%3A00%20/ip-addr
```

### Received message from host

```json
{
  "Cisco-IOS-XE-wireless-access-point-oper:ip-addr": "192.168.X.X"
}
```
