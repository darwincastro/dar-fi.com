## Cisco Platform

C9800-CL

## WLC's version used

17.6.2

## Yang Module

Cisco-IOS-XE-wireless-access-point-oper

## Prefix


/wireless-access-point-oper


## XPath


/wireless-access-point-oper:access-point-oper-data/capwap-data


## Example:
### Sending

```json
http://10.93.178.145:8480/restconf/proxy/https://10.93.178.68:443/restconf/data/Cisco-IOS-XE-wireless-access-point-oper:access-point-oper-data/capwap-data=00%3A04%3A05%3A00%3A02%3A00/ap-location
```

### Received message from host

```json
{
  "Cisco-IOS-XE-wireless-access-point-oper:ap-location": {
    "floor": 0,
    "location": "default-location",
    "aaa-location": {
      "civic-id": "",
      "geo-id": "",
      "oper-id": ""
    }
  }
}
```
