## Cisco Platform

C9800-CL

## CLI

```
show wireless stats client detail 
```

## WLC version used


17.9.3


## Yang Module


Cisco-IOS-XE-wireless-client-global-oper


## Prefix


/wireless-client-global-oper


## XPath


/wireless-client-global-oper:client-global-oper-data/client-global-stats-data


## Example:

### GET

```json
curl -X 'GET' \
  'http://<IP:PORT>/restconf/proxy/https://<IP:PORT>/restconf/data/Cisco-IOS-XE-wireless-client-global-oper:client-global-oper-data/client-dot11-stats/client-roaming-stats' \
  -H 'accept: application/yang-data+json'
```

### Sending

```json
http://<IP:PORT>/restconf/proxy/https://<IP:PORT>/restconf/data/Cisco-IOS-XE-wireless-client-global-oper:client-global-oper-data/client-dot11-stats/client-roaming-stats
```
